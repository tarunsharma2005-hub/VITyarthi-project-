*Clinic Appointment Booking System*
#Project Overview
	This is a simple, interactive Python application that allows clinic staff or patients to book, view, and cancel appointments via a 	command-line interface. Appointment data is saved locally in a JSON file to provide persistence between sessions.

#Features
	1)Book a new appointment with patient name, doctor name, date, time, and contact

	2)View all booked appointments

	3)Cancel an existing appointment

	4)Data is saved in a local JSON file for easy management

	5)Straightforward menu-driven text UI

#Tools Used
	1)Python 3.8+

	2)Standard Library: json, os, datetime

	3)No external dependencies required

#Installation and Setup
	1)Clone or download this project folder.

	2)Ensure you have Python 3.8 or newer installed.

	3)Place the provided Python script (clinic_booking.py) in your working directory.

# Run the script
python TARUN.py


#Instructions for Testing
	1)Book Appointment: Select option 1, fill out the prompted fields, and confirm.

	2)View Appointments: Select option 2. All appointments will display.

	3)Cancel Appointment: Select option 3, then enter the appointment number to cancel.

	4)Exit: Select option 4 to quit.

	5)Appointments are saved to appointments.json. You can open this file to verify bookings.

#Additional Notes
	1)No registration or login required.

	2)For a fresh test run, you can delete appointments.json from your directory.
